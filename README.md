# Clinical-Trial-Data-Analysis-for-a-new-diabetic-medication
Clinical Trial Data Analysis – Diabetic Medication  This project contains an **Exploratory Data Analysis (EDA)** of a clinical trial dataset for a new diabetic medication. The goal is to analyze patient responses, assess efficacy, and visualize key outcomes. 

## Objectives

- Analyze patient data from clinical trials  
- Evaluate efficacy of the medication on blood glucose levels  
- Identify patterns in patient responses  
- Visualize outcomes with charts and graphs  

## Tools Used

- Python (`pandas`, `numpy`, `matplotlib`, `seaborn`)  
- Jupyter Notebook for analysis  
- Excel for optional initial cleaning
